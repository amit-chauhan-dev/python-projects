# Number Guessing Game 

## 📌 Project Overview

The **Number Guessing Game** is a simple Python console-based project. In this game, the computer randomly selects a number between **1 and 10**, and the player must guess the correct number.

After each guess, the program gives a hint:

* If the guessed number is **too high**, it tells the user.
* If the guessed number is **too low**, it also informs the user.

The game continues until the player guesses the correct number.

---

## Concepts Used

This project is built using basic Python concepts:

* `random` module
* `while` loop
* conditional statements (`if`, `elif`, `else`)
* user input (`input()` function)

---

## How the Game Works

1. The computer generates a random number between **1 and 10**.
2. The user enters a number as a guess.
3. The program compares the guess with the secret number.
4. The program tells whether the guess is **Too High** or **Too Low**.
5. The game continues until the correct number is guessed.
