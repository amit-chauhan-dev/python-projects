# To-Do List App (Python Project)

> A simple and efficient **console-based To-Do List application** built using Python, allowing users to manage daily tasks with file storage support.

---

## Overview

The **To-Do List App** helps users organize their daily tasks by providing options to **add, view, and delete tasks**.
All tasks are saved in a file, ensuring data persistence even after the program is closed.

This project demonstrates practical usage of **lists and file handling** in Python.

---

## Key Features

* **Add Tasks**

  * Users can add new tasks to the list

* **View Tasks**

  * Displays all saved tasks in a numbered format

* **Delete Tasks**

  * Remove tasks by selecting their number

* **Persistent Storage**

  * Tasks are saved in a file (`tasks.txt`)

* **Interactive Menu**

  * Continuous loop-based user interaction

---

## Tech Stack

* **Language:** Python
* **Concepts Used:**

  * Lists
  * Loops & Conditionals
  * Functions
  * File Handling (`read`, `write`)

---

## How It Works

1. User selects an option from the menu
2. Tasks can be added, viewed, or deleted
3. Changes are automatically saved to a file
4. Data remains available after restarting the program
