# Contact Book (Python Project)

A simple and efficient **console-based Contact Book application** built using Python, allowing users to store, search, and manage contacts with persistent file storage.

---

## Overview

The **Contact Book** is a Python application designed to manage personal contacts.
Users can **add, view, search, and delete contacts**, and all data is stored in a file for future use.

This project demonstrates practical implementation of **dictionaries and file handling** in Python.

---

## Key Features

* **Add Contact**

  * Save new contact with name and phone number

* **View Contacts**

  * Display all saved contacts in a structured format

* **Search Contact**

  * Find a contact by name instantly

* **Delete Contact**

  * Remove existing contact 

* **Persistent Storage**

  * Contacts are saved in a file (`contacts.txt`)

* **Interactive Menu**

  * Continuous loop-based user interaction

---

## Tech Stack

* **Language:** Python
* **Concepts Used:**

  * Dictionaries
  * Loops & Conditionals
  * Functions
  * File Handling (`read`, `write`)

---

## How It Works

1. User selects an option from the menu
2. Can add new contacts (name + phone)
3. Search contacts by name
4. Delete contacts if needed
5. All changes are saved automatically in a file

---

## Learning Outcomes

Through this project, you will learn:

* How to use dictionaries for data storage
* How to perform CRUD operations (Create, Read, Update, Delete)
* How to implement file-based data persistence
