# Password Generator (Python)

## 📌 Project Overview

The Password Generator is a simple Python program that automatically creates strong and random passwords.
The generated password includes letters, numbers, and special symbols to make it secure.

This project is useful for beginners to understand how random values and strings work in Python.


## Concepts Used

* Python `random` module
* Python `string` module
* loops
* user input


## How It Works

1. The user enters the desired password length.
2. The program combines letters, digits, and special characters.
3. A random password is generated using these characters.
4. The generated password is displayed to the user.
