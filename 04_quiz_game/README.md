# Quiz Game

> A feature-rich **console-based Quiz Game** built using Python, demonstrating core programming concepts with real-world logic and clean structure.


## Overview

This project is an interactive **Multiple Choice Quiz Game** where users answer questions within a time limit.
It tracks performance, calculates scores, and maintains a persistent **high score system**.

The goal of this project is to apply Python fundamentals in a practical and structured way.


## Key Features

**User Interaction**

  * Personalized welcome with user name

**Dynamic Gameplay**

  * Questions are shuffled randomly every time

**Timer-Based Questions**

  * Each question must be answered within a fixed time limit

**Multiple Choice Questions (MCQ)**

  * Options: A / B / C / D

**Score Tracking**

  * Displays final score at the end

**High Score System**

  * Saves and loads highest score using file handling


## Tech Stack

* **Language:** Python
* **Concepts Used:**

  * Lists & Dictionaries
  * Loops & Conditionals
  * Functions (logic structuring)
  * File Handling
  * Modules (`random`, `time`)


## Learning Outcomes

Through this project, you will understand:

* How to structure real-world Python programs
* How to manage user input and validation
* How to implement timers and randomness
* How to store persistent data using files
